#  Sales Data Analysis using Python

##  Objective

This project analyzes one year of retail sales data to uncover business trends, customer purchasing behavior, and product performance using Python.

## 🛠 Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

##  Dataset

The dataset contains customer purchase records, including:

- Order ID
- Product
- Quantity Ordered
- Price Each
- Order Date
- Purchase Address

##  Analysis Performed

- Data Cleaning
- Feature Engineering
- Monthly Sales Analysis
- City-wise Sales Analysis
- Best Selling Products
- Sales by Hour
- Products Bought Together
- Correlation Analysis

##  Key Insights

- December recorded the highest sales revenue.
- San Francisco generated the highest revenue among all cities.
- AAA Batteries (4-pack) was the best-selling product.
- Customer purchases peaked around 7 PM.
- Frequently purchased product pairs provide opportunities for bundle promotions.

##  Business Recommendations

- Increase inventory before peak sales months.
- Focus marketing efforts on high-performing cities.
- Promote complementary products together.
- Optimize staffing during peak shopping hours.
- Use sales data for demand forecasting.
